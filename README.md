### Features

- [x] Sort to dos in (ascending or descending order)
- [x] Add a new to do
- [x] Delete a to do
- [x] Select a random to do
- [x] Display number of to dos
- [x] Delete all to dos
- [x] Input validation and delete confirmation popups
